1번째 : 프렌차이즈  7-ELENEN
URL:https://www.7-eleven.co.kr/


단점
- 웹표준 미준수
	XHTML 1.0 Transitional 사용 (구식 마크업)
	테이블 기반 레이아웃으로 구조적 한계

-사용자 경험(UX) 문제
	복잡한 메뉴 구조로 원하는 정보 찾기 어려움
	이벤트 정보가 작은 배너로만 제공
	매장 찾기 기능이 직관적이지 않음

-비주얼 디자인 한계
	상품 이미지들이 작고 매력적이지 않음
	브랜드 아이덴티티가 웹에서 제대로 표현 안됨




2번째 :  한국중부발전 
URL: https://www.komipo.co.kr/kor/main/main.do

단점
-정보 과부하
	메인페이지에 너무 많은 정보가 한 번에 노출
	12개의 메인 메뉴 + 수많은 아이콘들로 사용자 혼란 유발
	핵심 CTA(Call To Action) 버튼이 명확하지 않음

-성능 및 유지보수성
	다수의 개별 스크립트 파일 로딩 (add.js, search.js, xmlhttp.js 등)
	인라인 스타일과 외부 CSS 혼재 사용
	불필요한 shadow div들 (shadow, shadow2, shadow3) - 현대적 CSS로 대체 가능

- 구식 디자인 언어
	2010년대 초반 스타일의 레이아웃과 색상 조합
	플랫 디자인이나 현대적 UI 트렌드 미반영
	카드형 레이아웃 부재로 정보 구조화 부족

-사용자 경험 취약점
	메인 슬라이더가 너무 크고 내용 파악이 어려움
	아이콘들의 의미가 직관적이지 않음 (특히 하단 12개 서비스 아이콘)
	지도와 텍스트가 혼재되어 가독성 저하

-브랜딩 및 신뢰도
	공기업임에도 불구하고 전문성과 신뢰성이 시각적으로 전달되지 않음
	한국중부발전의 친환경 에너지 기업 이미지가 약함
	경쟁사 대비 낙후된 이미지로 인한 브랜드 가치 하락
