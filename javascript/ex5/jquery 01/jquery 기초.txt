    7) 배열관리 => $.each() 메서드
        $(document).ready(function () {
            var array = [
                { name: 'Hanbit Media', link: 'http://hanb.co.kr' },
                { name: 'Naver', link: 'http://naver.com' },
                { name: 'Daum', link: 'http://daum.net' },
                { name: 'Paran', link: 'http://paran.com' }
            ];

            $.each(array, function (index, item) {
                var output = '';
                output += '<a href="' + item.link + '">';
                output += ' <h1>' + item.name + '</h1>';
                output += '</a>';
                document.body.innerHTML += output;
            });
        });

         ex3) filter() 메소드   =>  (선택자).filter.(function(){ });
          $(document).ready(function () {
            $('h3').filter(function (index) {
                return index % 3 == 0;
            }).css({

        ex4) filter()을 사용하면 문서 객체의 범위를 좁혀 선택할 수 있다

   ex7) append() / prepend() / after() / before() 메소드
       $(document).ready(function () {
            var h1 = '<h1>Header1</h1>';
            var h2 = '<h2>Header2</h2>';

            $('body').append(h1, h2, h1, h2);
        });

        * $(A).append(B)  => B를 A의 뒷 부분에 추가
          $(A).prepend(B)  => B를 A의 앞 부분에 추가
          $(A).after(B)  => B를 A의 뒷 부분에 추가
          $(A).before(B)  => B를 A의 뒷 부분에 추가
